インストール方法

イメージの利用にはOracle Virtual Boxが必要です。

  • https://www.virtualbox.org/

VirtualBoxに .ova ファイルをインポートします。 ほとんどのユーザーにとっては、初期設定で問題ありませんが、パフォーマンスに問題がある場合は変更が必要かもしれません。

ゲストOSには Ubuntu 14.04 を使用します。

イメージ内の初期アカウントは次の通りです：

  • username: frappe
  • password: frappe
  • mysql root password: frappe

VM起動後、ホストOSのブラウザにて次のURLにアクセスします：

  • http://localhost:8080

ログイン情報は次の通りです：

  • user: Administrator
  • password: admin

イメージアップデート

VM内のERPNextをアップデートする場合、Virtual Boxのシェル (username: "frappe", password: "frappe") にログインし、次の通り実行します。 

cd frappe-bench
bench update

問題が発生した場合は、 https://discuss.erpnext.com にてIssueを登録してください。
